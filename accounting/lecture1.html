مثال Python:

Python Example:


# Calculate compound interest
def compound_interest(principal, rate, time, n):
    """
    Calculate compound interest
    principal: Initial amount
    rate: Annual interest rate (decimal)
    time: Time in years
    n: Number of times interest is compounded per year
    """
    amount = principal * (1 + rate/n)**(n*time)
    interest = amount - principal
    return round(interest, 2)

# Example usage
principal = 50000
rate = 0.125
time = 5
n = 12  # Monthly compounding

result = compound_interest(principal, rate, time, n)
print(f"Future Value: ${principal + result:,.2f}")
print(f"Interest Earned: ${result:,.2f}")

مثال JavaScript:

JavaScript Example:


// Calculate NPV (Net Present Value)
function calculateNPV(rate, cashFlows) {
  return cashFlows.reduce((npv, cf, i) => {
    return npv + cf / Math.pow(1 + rate, i);
  }, 0);
}

// Example: Investment analysis
const discountRate = 0.1;  // 10%
const cashFlows = [-100000, 30000, 35000, 40000, 45000];

const npv = calculateNPV(discountRate, cashFlows);
console.log(`NPV: $${npv.toFixed(2)}`);

if (npv > 0) {
  console.log("✅ Investment is profitable!");
} else {
  console.log("❌ Investment is not profitable.");
}

📘 مبدأ الاستحقاق

📘 Accrual Principle

تسجيل الإيرادات والمصروفات عند حدوثها وليس عند استلام أو دفع النقدية.

Recording revenues and expenses when they occur, not when cash is received or paid.

💚 مبدأ المطابقة

💚 Matching Principle

مطابقة المصروفات مع الإيرادات المرتبطة بها في نفس الفترة المحاسبية.

Matching expenses with related revenues in the same accounting period.

❤️ مبدأ التكلفة التاريخية

❤️ Historical Cost Principle

تسجيل الأصول بتكلفة الحصول عليها وليس بقيمتها السوقية الحالية.

Recording assets at their acquisition cost, not current market value.

💜 Going Concern

افتراض أن المنشأة ستستمر في العمل لفترة طويلة في المستقبل.

Assuming the business will continue operating for the foreseeable future.
